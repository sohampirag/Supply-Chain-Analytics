# 📊 Supply Chain Analytics using Python, SQL & Power BI

## 📌 Project Overview

This project demonstrates an end-to-end Supply Chain Analytics solution using Python, SQL, and Power BI. 
The objective is to analyze supply chain data, uncover business insights, and build interactive dashboards to support data-driven decision-making.


## 🎯 Objectives

- Perform data cleaning and preprocessing
- Build an ETL pipeline using Python
- Analyze sales, profit, and delivery performance
- Generate business insights using SQL
- Create an interactive Power BI dashboard
- Monitor key supply chain KPIs

---

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- SQL (MySQL)
- Power BI
- VS Code

---

## 📂 Project Workflow

1. Data Extraction
2. Data Cleaning
3. Data Transformation
4. Feature Engineering
5. SQL Analysis
6. Dashboard Development
7. Business Insights

---

## 📈 Key Performance Indicators (KPIs)

- Total Sales
- Total Profit
- Profit Margin
- Order Quantity
- Delivery Performance
- Late Delivery Risk
- Market-wise Sales
- Category-wise Sales


---

## 🚀 Key Features

- Built an ETL pipeline using Python and Pandas
- Cleaned and transformed 180K+ supply chain records
- Performed Exploratory Data Analysis (EDA)
- Created business insights using SQL
- Developed an interactive Power BI dashboard
- Generated KPI reports for business decision-making
